**Communication solution**

Harbinger is a powerful platform designed to optimize mass communication via WhatsApp and SMS. With it, you can create and send personalized messages to large volumes of contacts efficiently and effectively.

Its intuitive interface makes it easy to manage contact lists, allowing you to create, organize, and update them with ease.

- **Mass sending of WhatsApp and SMS messages**. Create individualized and personalized messages.
- **Easily manage contact lists**. Create contact lists.
- **Schedule sending times** with customizable frequencies.

## Features
- **Personalize your messages** Use each contact's personal information with text variables. Create personalized templates and reuse them
- **Scheduling** Set up messages to be sent at predefined times or create a schedule with repeats.
- **Triggers** Send messages based on user actions
- **Integration** Connect with your business tools
- **Sequences** Create interconnected message flows
