En esta sección podrás gestionar la incorporación de nuevos clientes y mantenerlos informados sobre las últimas novedades de cada producto.

Desde aquí podrás dar de alta nuevos entornos, registrar la información de contacto y enviar automáticamente las release notes correspondientes a cada cliente.

![Admin](/img/admin/portada.png)

## Alta de clientes

Completa este formulario para registrar un nuevo cliente dentro del portal.

Los datos ingresados se utilizarán para crear su entorno y habilitar el envío de comunicaciones y notas de lanzamiento personalizadas.

<iframe className="frame" width="100%" height="800px" src="https://n8n.seidoranalytics.com/form/55d06321-b6ee-4217-8dab-6ded1ac46324" />


## Release Notes
Utiliza este formulario para enviar las release notes más recientes a los clientes registrados.

Podrás seleccionar el producto, versión y adjuntar el mensaje correspondiente antes de enviar las notificaciones.

<iframe className="frame" width="100%" height="800px" src="https://n8n.seidoranalytics.com/form/55d06321-b6ee-4217-8dab-6ded1ac46324" />
