## Infraestructura Seidor Analytics
La siguiente tabla detalla la inversión requerida para implementar **Act·in | SAP S/4HANA Analytics**, considerando distintos escenarios según el número de módulos. Incluye costos de suscripción mensual, licencias, componentes de extracción de datos, horas de soporte y el valor de horas adicionales.

| Concepto                                | 1 Módulo      | 2 Módulos     | 3 Módulos     |
|-----------------------------------------|---------------|---------------|---------------|
| **Suscripción Mensual (*)**             | 3.000 USD     | 3.500 USD     | 4.000 USD     |
| **Licencias Tableau (viewer) (*)**      | 5             | 5             | 5             |
| **Theobald Universal Xtractor (*)**     | 1 Server <br/> 1 Destination <br/> 100 extracciones | 1 Server <br/> 1 Destination <br/> 100 extracciones | 1 Server <br/> 1 Destination <br/> 100 extracciones |
| **Horas Hombre Soporte / Mejora Incluidas** | 24            | 40            | 64            |
| **Valor Hora Adicional**                | 45 USD        | 45 USD        | 45 USD        |


**Notas**
- Este escenario de inversión asume que el cliente no utiliza su propia cuenta de AWS para aprovisionar los servicios.​
- (*) Contrato de suscripción con duracion de 3 años, con facturación mensual adelantada.​
- Valores expresados en dólares americanos, no contienen impuestos ni retenciones.​
- No se incluyen gastos de transporte, viaje o estadía.
