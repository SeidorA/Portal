## Funcionalidades Principales

Las funcionalidades definen la esencia de cada plataforma: qué puede hacer y cuál es su enfoque principal. En este apartado se comparan las capacidades de creación de asistentes virtuales e integración con flujos de trabajo empresariales.

| Característica                  | Daiana                                                                                                                                         | Copilot Studio (Microsoft)                                                                                                                          |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Funcionalidades Principales** | Creacion y gestión de asistentes virtuales basados en IA. Enfoque empresarial para acceso a informacion privada y bases de datos corporativas. | Creación de bots sin código, integración con Power Platform, generación de flujos de trabajo y soporte para automatización de tareas empresariales. |

---

##  Ecosistema

El ecosistema determina el nivel de flexibilidad de la herramienta: si se limita a un entorno o si permite integrarse con distintas tecnologías.

| Característica | Daiana                                                                                                         | Copilot Studio (Microsoft)                                                          |
| -------------- | -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| **Ecosistema** | Agnóstico. Se puede integrar con diferentes nubes y ofrece conectividad con distintos tipos de bases de datos. | Dependencia del ecosistema Microsoft para aprovechar al máximo las funcionalidades. |

---

## Facilidad de Uso

La facilidad de uso está ligada al autoservicio, la autonomía de los equipos y la necesidad de contar con conocimientos técnicos para sacar provecho de la herramienta.

| Característica       | Daiana                                                                                                                                | Copilot Studio (Microsoft)                                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Facilidad de Uso** | Alta. Orientada al autoservicio para que los usuarios no técnicos puedan crear y entrenar sus asistentes virtuales de forma autónoma. | Centrado en usuarios no técnicos, pero requiere conocimiento del ecosistema Microsoft (Power Platform, Microsoft 365, Dynamics 365) para un aprovechamiento completo. |

---

##  Bases de Datos

El nivel de integración con bases de datos corporativas marca la diferencia en escenarios empresariales donde la información es el centro de la operación.

| Característica     | Daiana                                                                                       | Copilot Studio (Microsoft)                                             |
| ------------------ | -------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **Bases de datos** | Integración nativa con PostgreSQL, Snowflake, MySQL, SQL Server, Amazon Redshift y SAP Hana. | No existe integración nativa con bases de datos (requiere desarrollo). |

---

##  Soporte y Actualizaciones

Un buen soporte y la frecuencia de actualizaciones son claves para garantizar continuidad, seguridad y evolución de la herramienta.

| Característica                | Daiana                                                                              | Copilot Studio (Microsoft)                                        |
| ----------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| **Soporte y Actualizaciones** | Soporte empresarial, actualizaciones programadas, enfoque en clientes corporativos. | Actualizaciones regulares, integración con la suite de Microsoft. |

---

##  Modelos LLM Soportados

La capacidad de soportar múltiples modelos determina la flexibilidad, escalabilidad y personalización de la solución.

| Característica             | Daiana                                                                                                                                                           | Copilot Studio (Microsoft)                                                                                          |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Modelos LLM Soportados** | GPT-4o, GPT-4, GPT-3.5, GPT-3, AWS Bedrock, NIBtensorsLLM, Cohere, Google PaLM, Google Vertex AI, HuggingFace Inference, Ollama, OpenAI, Replicate (en roadmap). | Modelos de Azure OpenAI (GPT-4, GPT-3). Permite integrar modelos personalizados entrenados con datos de la empresa. |

---

##  Costo

El costo es un factor decisivo, no solo por el valor monetario, sino por el modelo de consumo y la relación con el volumen de uso.

| Característica | Daiana           | Copilot Studio (Microsoft)                                             |
| -------------- | ---------------- | ---------------------------------------------------------------------- |
| **Costo**      | Licenciamiento basado en cantidad de mensajes: Small (USD 1.000 x 10.000 mensajes/mes), Medium (USD 1.800 x 20.000 mensajes/mes), Large (consultar). |USD200 x 25.000 mensajes/mes  Tasas de consumo:<br/> 1) Regular (IA no generativa) = 1 mensaje. <br/> 2) IA Generativa (Respuestas sobre tus datos) = 2 mensajes. |

## Desventajas Copilot Studio (Microsoft)
- Dependencia del ecosistema Microsoft para aprovechar al máximo las funcionalidades.
- El producto es una plataforma para crear bots, con algunas funcionalidades de AI. No es un producto AI native.
