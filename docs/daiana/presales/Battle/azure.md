## Funcionalidades Principales

Las funcionalidades definen la esencia de cada plataforma: qué puede hacer y cuál es su enfoque principal. En este apartado se comparan las capacidades de creación de asistentes virtuales e integración con flujos de trabajo empresariales.

| Característica                  | Daiana                                                                                                                                         | Copilot Studio (Microsoft)                                                                                                                          |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Funcionalidades Principales** | Creación y gestión de asistentes virtuales basados en IA. Enfoque empresarial para acceso a información privada y bases de datos corporativas. | Creación de bots con soporte para múltiples canales, integración con servicios de Azure |

---

##  Ecosistema

El ecosistema determina el nivel de flexibilidad de la herramienta: si se limita a un entorno o si permite integrarse con distintas tecnologías.

| Característica | Daiana                                                                                                         | Copilot Studio (Microsoft)                                                          |
| -------------- | -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| **Ecosistema** | Agnóstico. Se puede integrar con diferentes nubes y ofrece conectividad con distintos tipos de bases de datos. | Integración nativa con Azure, Microsoft 365.|

---

## Facilidad de Uso

La facilidad de uso está ligada al autoservicio, la autonomía de los equipos y la necesidad de contar con conocimientos técnicos para sacar provecho de la herramienta.

| Característica       | Daiana                                                                                                                                | Copilot Studio (Microsoft)                                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Facilidad de Uso** | Alta. Orientada al autoservicio para que los usuarios no técnicos puedan crear y entrenar sus asistentes virtuales de forma autónoma. | Alta (dependiendo del uso de otros servicios de Azure). |

---

##  Bases de Datos

El nivel de integración con bases de datos corporativas marca la diferencia en escenarios empresariales donde la información es el centro de la operación.

| Característica     | Daiana                                                                                       | Copilot Studio (Microsoft)                                             |
| ------------------ | -------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **Bases de datos** | Integración nativa con PostgreSQL, Snowflake, MySQL, SQL Server, Amazon Redshift y SAP Hana. | No existe integracion nativa con base de datos (requiere desarrollo).|

---

##  Soporte y Actualizaciones

Un buen soporte y la frecuencia de actualizaciones son claves para garantizar continuidad, seguridad y evolución de la herramienta.

| Característica                | Daiana                                                                              | Copilot Studio (Microsoft)                                        |
| ----------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| **Soporte y Actualizaciones** | Soporte empresarial, actualizaciones programadas, enfoque en clientes corporativos. | Soporte a través de Azure, actualizaciones regulares, comunidad activa. |

---

##  Modelos LLM Soportados

La capacidad de soportar múltiples modelos determina la flexibilidad, escalabilidad y personalización de la solución.

| Característica             | Daiana                                                                                                                                                           | Copilot Studio (Microsoft)                                                                                          |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Modelos LLM Soportados** | GPT-4o, GPT-4, GPT-3.5, GPT-3, AWS Bedrock, NIBtensorsLLM, Cohere, Google PaLM, Google Vertex AI, HuggingFace Inference, Ollama, OpenAI, Replicate (en roadmap). | Similar a Copilot Studio, Azure Bot Service soporta modelos de la suite Azure OpenAI como GPT-4 y GPT-3. Además, permite la integración de modelos personalizados creados a través de Azure Machine Learning para casos de uso específicos. |

---

##  Costo

El costo es un factor decisivo, no solo por el valor monetario, sino por el modelo de consumo y la relación con el volumen de uso.

| Característica | Daiana           | Copilot Studio (Microsoft)                                             |
| -------------- | ---------------- | ---------------------------------------------------------------------- |
| **Costo**      | Licenciamiento basado en cantidad de mensajes: Small (USD 1.000 x 10.000 mensajes/mes), Medium (USD 1.800 x 20.000 mensajes/mes), Large (consultar). | Costos basados en el uso de Azure, con modelos flexibles de pago. |

## Desventajas Copilot Studio (Microsoft)
Puede ser costoso dependiendo del uso de otros servicios de Azure, más técnico que otros competidores.
